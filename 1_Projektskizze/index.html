1. Projekt-Beschreibung

In diesem Projekt wird in einer Gruppe von drei Personen ein Schere-Stein-Papier-Spiel in Java programmiert, das die unten aufgeführten Anforderungen erfüllt.
Der Zeitrahmen beträgt die ersten 14 Semesterwochen, wobei in Woche 4 die Projektskizze abzugeben ist und in Woche 14 das Projekt fertiggestellt sein und vorgestellt werden muss.

2. Features und Anforderungen

Funktionale Anforderungen:

  • Das Spiel muss online von Spielern an verschiedenen Computern ausgeführt werden können
  • Es muss eine Highscore-Liste geben
  • Der Gewinner muss gespeichert werden
  • Schere, Stein oder Papier wird ausgewählt durch klicken auf das entsprechende Icon oder durch Shortcuts auf der Tastatur
  • Nach Ablauf eines Zeitlimits werden die Icons von beiden Spielern und der entsprechende Gewinner angezeigt
  • Bei gleicher Auswahl wird „unentschieden“ angezeigt
  • Die Spieler können ihren Namen wählen
  • Das Programm zeigt an, welche Spieler online sind. Man kann aus einer Liste wählen, gegen wen man spielen möchte

Nicht-funktionale Anforderungen:

  • Es soll eine Menüoberfläche geben, von der aus der andere Spieler gesucht und das Spiel gestartet werden kann, sowie das Leaderboard angezeigt wird
  • Es soll eine Spieloberfläche geben, auf der die drei Spielfiguren ausgewählt werden können
  • Die Bilder der Spielfiguren sollen auf den Buttons angezeigt werden
  • Das Programm soll als ausführbares Programm auf zwei verschiedenen Computern laufen
  • Zur programmierung wird die neueste Version von Eclipse verwendet und es wird in der Java 17 SDK programmiert

Das spiel besteht aus drei Klassen. In der Klasse "Spieler" werden alle wichtigen Informationen gespeichert, auch die, die von anderen Klassen benögtigt werden.
Die Klasse "Spiel" ist dafür verantwortlich, dass das Spiel gestartet wird, zeigt einen Countdown an, während welchem beide Spieler ihre Figur wählen müssen, und ist für die Auswertung des Ergebnisses zuständig.
Die Klasse "Leaderboard" trägt die relevanten Werte der Spieler zusammen und ordnet sie entsprechend der Siege.
